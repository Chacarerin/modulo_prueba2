# 🚀 Módulo de Prueba Básico — Python

[![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)](https://python.org)

Repositorio de configuración y prueba inicial del entorno de desarrollo Python. Actúa como sandbox de validación del ecosistema antes de iniciar proyectos de mayor envergadura.

## 🧠 Contexto Pedagógico y Teórico
La gestión del entorno de desarrollo es una habilidad crítica en la ingeniería de software. Este repositorio documenta el proceso de validación inicial del entorno de Python: verificar que el intérprete está correctamente instalado, que las variables de entorno son accesibles, y que el flujo básico de ejecución de scripts funciona correctamente. Este paso, aunque simple, es el fundamento de la reproducibilidad de entornos, un pilar del desarrollo profesional.

## ⚙️ Tecnologías y Metodologías Aplicadas
* **Python Estándar**: Prueba de ejecución básica del intérprete de Python validando la asignación de variables y la función de impresión estándar.
* **Control de Versiones Git**: El propósito principal del repositorio es validar el flujo de trabajo `git init → commit → push`, siendo un banco de pruebas para el control de versiones.
